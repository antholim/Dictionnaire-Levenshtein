## Dictionnaire Levenshtein
Demo : https://youtu.be/Ym1r3lEZPxc <br>
This project was built for the course Object-Oriented Programming and Data Structure and Algorithms. This utilizes the Levenshtein Algorithm and calculates the shortest distance between the word entered and the array of words in the .txt file.

This was built using Java JDK 17.0.1
